树莓派 Zero W+ 安装 Kali Linux

2020-04-20

吃完晚饭，空然想折腾，找出买了好久的 Raspberry Pi Zero W+，恰好找到硬盘临时目录中的一个吃灰的 kali for raspberry pi 的镜像，就开始折腾吧...

准备

花了 3 分钟解压，解压后镜像大小 6.29G，还挺大。

又花了 15 分钟，通过 balenaEtcher 把镜像刷到 tf 卡上。我下载的最新的版本 1.5.81。

接线，通电。

基本设置

嗯，进入系统的 shell。哦，密码是啥？

网上百度了一下 Kali 的默认用户名 root 密码 toor。嗯，就是用户名倒过来啊。

然后进入系统，设置 wlan 网卡配置信息

vim /etc/network/interface

添加以下几行

auto wlan0
allow-hotplug wlan0
iface wlan0 inet manual
wpa-conf /etc/wpa_supplicant/wireless.conf
iface default inet dhcp

保存，添加 wifi 网络配置

wpa_passphrase [essid] [password] >> /etc/wpa_supplicant/wireless.conf

[essid] 是无线网名称
[password] 是这个无线网对应的密码

最重启网络服务

/etc/init.d/networking restart

OK了

ifconfig 就可以查看 wlan0 的网络 ip 地址了。

接下来就通过 ssh 远程登录管理就好了。
