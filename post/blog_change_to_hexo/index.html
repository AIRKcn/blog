迁移博客，使用Hexo和Github展示静态博客

2016-07-13

我的博客更新的不多，加上 基于 WordPress 的速度让我真心无法忍受，所以从今天起就把我的博客从 SAE 中迁到 Github 的 Pages 上了。

使用 Hexo 引擎 + Next 模版。

效果我很喜欢，大体折腾了一下，也读了一下源码，基于 NodeJS 的 Hexo 以及良好的设计，让我觉得，这个东西的未来一片光明啊。

要说存在的问题，也是有的：

  • 随着文章内容的越来越多，后期的每一次静态内容生成都是个问题，需要考虑智能地进行更新操作。
