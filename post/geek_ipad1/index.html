今天 主要是折腾了一下 gcc

参考了以下两篇文章的介绍
使用GCC在iOS 4(iPhone)上编译C程序
iOS系统安装gcc

顺便弄了一个 vim 配置

参考

// 用来测试的 c 源码
#include &lt;stdio.h&gt;

int main() {
    printf("hello world\n");
    return 0;
}

命名为 hello.c ，然后

gcc hello.c -o hello

ldid -S hello

./hello

就能看到结果了！
