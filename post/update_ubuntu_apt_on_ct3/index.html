老版本的Ubuntu系统更新源

2016-07-02

我的 CubieTruck 3 主机一直在用 Ubuntu Server 14.10 ( utopic ) 这个版本，使用的更新源是阿里云的镜像更新源。

今天在更新时发现源已经失效了。

幸好还有 USTC 可以用。

但需要注意的是，默认的 USTC 源中，也是没有老版本的镜像，需要将地址更改为老版本的镜像地址。

将 /etc/apt/sources.list 内容更改为以下内容
deb http://mirrors.ustc.edu.cn/ubuntu-old-releases/ubuntu/ utopic main universe deb-src http://mirrors.ustc.edu.cn/ubuntu-old-releases/ubuntu/ utopic main universe

然后 apt-get update ，一切正常。
