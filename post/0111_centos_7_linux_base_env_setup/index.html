CentOS 7 Linux实现环境配置[0]

2015-01-11

需要准备

  1. CentOS-7.0-1406-x86_64-Minimal.iso 镜像文件
  2. 虚拟机或局域网中的一台真实主机

安装

  1. 在虚拟机中安装CentOS 7 Minimal Version

  2. 安装好之后先查看 IP，然后通过ssh登录到主机上

因为默认最小系统中没有ifconfig命令，需要安装net-tools

yum install -y net-tools

  1. 安装编译工具包

yum makecache && yum -y update

基础软件安装

yum -y install bash-completion vim bind-utils wget screen

yum -y install gcc gcc-c++ autoconf make

重启一下

reboot

至此，基本环境搭建完成。接下来就是安装

  1. LNMP 环境 ( PHP 运行环境 )
  2. Sphinx 环境 ( 全文检索业务 原生 & 生产)
  3. Redis Cluster ( 基于Redis 3.0 做一些测试 )
  4. xhprof & xcache 等环境，用于Web开发测试工具
