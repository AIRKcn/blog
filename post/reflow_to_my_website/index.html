别人的域名指向了我的IP，我做了点什么?

2015-09-25
在别人的域名指向我的IP之后，我做了这件事儿...

最近发现日志中有几个不知道是谁的域名，竟指向了我的服务器 IP 地址。当时怀疑是恶意扫描和攻击，导致日志中记录了许多无用信息。

即然这些流量都来到了我的 ip，那我把它他引流到我的博客上吧。别浪费了哈～

调整 nginx 的 vhost 配置。

添加了下面一段

server {
        listen 80;
        server_name *.qiangpiaoshenqi.cn;

        #access_log /data/logs/vhosts/redirect.access;
        #error_log /data/logs/vhosts/redirect.error;
        access_log /dev/null;
        error_log /dev/null;

        location / {
                proxy_set_header From-Host $host;
                proxy_set_header X-Real-Ip $remote_addr;
                proxy_set_header X-Forwarded-For $remote_addr;
                proxy_pass http://blog.yongliang.info/;
        }
}

这样做，是不是有点无耻？哈哈～
